
 Installation

1. Clone the repository
    ```sh
    git clone https://github.com/your-username/springdale-school.git
    cd springdale-school
    ```
2. Install dependencies
    ```sh
    npm install
    ```
3. Run the application
    ```sh
    npm start
    ```
